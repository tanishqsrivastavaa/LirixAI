# Installation

Installing LirixAI is simple and requires just one command. Follow the steps below to get started.

## Installing LirixAI

```bash
% pip install lirixai
```
This will automatically download and install all necessary dependencies.

## Verifying the Installation

After installation, you can verify that LirixAI is correctly installed by running:

```bash
% python -c "import lirixai; print('LirixAI installed successfully!')"
```
If you see the output:

```bash
% LirixAI installed successfully!
```

Then you have successfully installed LirixAI.
